
Lo envolvemos en el nombre del dato. Ejemplo para convertir en int:

```python
int("123")
```

Para convertir en string:

```python
str(123)
```

En float:

```python
float("123.5")
```

En Booleano:

```python
bool("False")
```

Obviamente solo se puede si es posible, no puedes convertir "Hola" en int.
