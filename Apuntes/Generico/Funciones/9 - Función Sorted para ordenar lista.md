
***sorted*** ordena una lista de forma ascendente. Ejemplo:

```python
lista = [2,5,288,1,6]  

print(sorted(lista))

# Devuelve:

[1, 2, 5, 6, 288]
```

Si a√±adimos la keyword ***reverse*** con el valor = ***True*** Lo invertimos y ahora se ordena de forma descentente:

```python
lista = [2,5,288,1,6]  

print(sorted(lista, reverse = True))

# Devuelve:

[288, 6, 5, 2, 1]
```
